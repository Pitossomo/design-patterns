5 Design Patterns Mais Usados em Engenharia de Software

Padrão de Projeto Singleton

O Padrão de Projeto Singleton é uma abstração que representa a classe como um objeto único.

Padrão de Projeto Factory

O Padrão de Projeto Factory é uma abstração que representa a criação de objetos sem especificar sua classe concreta.

Padrão de Projeto Observer

O Padrão de Projeto Observer é uma abstração que permite a notificação de mudanças em objetos.

Padrão de Projeto Strategy

O Padrão de Projeto Strategy é uma abstração que permite a escolha de algoritmos em execução.

Padrão de Projeto Template Method

O Padrão de Projeto Template Method é uma abstração que define um método comum para suas subclasses.
